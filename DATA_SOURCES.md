# 資料來源

本資料包整合多個公開或可授權使用的資料來源。每次發布應在 `manifest.json` 與資料包內的 `mobile_data_source_versions` 表保留來源版本與授權註記。

## P0：免費版核心資料

| 來源 | 用途 | 授權/注意事項 |
| --- | --- | --- |
| TFDA 全部藥品許可證資料集 | 台灣藥品中文名、英文名、許可證、劑型、適應症等 | 依政府資料開放授權條款使用；正式發布需保留來源與更新日期 |
| TFDA 藥品詳細處方成分資料集 | 成分、規格、含量與正規化 | 依政府資料開放授權條款使用；用於成分比對，不作為完整交互作用裁決 |
| TFDA ATC / AHFS 藥理治療分類資料 | 藥理分類、同類藥重複與規則比對基礎 | 分類輔助用，不能單獨代表禁忌或安全 |
| 健保署健保用藥品項查詢項目檔 | 健保碼、中英文名、成分、規格、廠商 | 需保留來源與版本；正式商用前仍應確認欄位使用條款 |

## P1：高價值規則與仿單資料

| 來源 | 用途 | 授權/注意事項 |
| --- | --- | --- |
| RxNorm / RxNav API | 國際藥名與 RxCUI 對應 | 免費官方 API；主要以美國資料為主，台灣商品名仍需 TFDA/NHI mapping |
| DailyMed API | 美國 SPL 仿單、禁忌、警語、交互作用欄位 | 免費官方資料；內容不等於台灣仿單 |
| openFDA | Label、FAERS、recall 輔助查詢 | 免費 API；不良事件不能直接代表因果關係 |
| FDA CYP / Transporter tables | CYP/P-gp/OATP 等規則庫來源 | 適合作為藥師審核規則的來源之一 |
| STOPP/START v3 | 老人多重用藥與不適當處方提示 | Open Access / CC BY 4.0；須保留引用與版本 |
| Phansalkar / ONC high-priority DDI list | 高優先交互作用規則起點 | 需保留論文與專家共識來源 |

## P2：補充資料

| 來源 | 用途 | 授權/注意事項 |
| --- | --- | --- |
| TFDA 藥品外觀資料 | 藥丸外觀與辨識輔助 | MVP 不以藥丸照片作唯一判斷 |
| MedlinePlus Connect | 病人友善衛教連結 | 主要英文/西文；中文需另處理 |
| DrugCentral | 成分、藥理與適應症輔助 | Open-access；需確認引用方式 |
| AGS Beers Criteria 2023 | 老人高風險藥物提示 | 授權需特別確認，不宜直接完整重製 |

## 明示限制

- 本資料包不是 Lexicomp、Micromedex、First Databank 或其他商用 DDI 資料庫的替代品。
- 免費版若資料不足，必須保守提醒「請詢問藥師或醫師」。
- 商用或醫療機構版可使用 BYOK 或院所後端 proxy 連接已授權資料庫。
